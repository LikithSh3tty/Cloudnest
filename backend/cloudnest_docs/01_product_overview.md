# CloudNest — Product Overview

CloudNest is a cloud storage and automated backup service for individuals and small teams. It syncs files across devices, keeps version history, and runs scheduled backups of selected folders.

## Core Features
- Cross-device sync (Windows, macOS, Linux, iOS, Android)
- Automatic versioning — keeps last 30 versions of any file for 90 days
- Scheduled folder backups (hourly, daily, weekly)
- Shared folders with role-based permissions (Owner, Editor, Viewer)
- End-to-end encrypted vaults (optional, opt-in per folder)

## Plans
CloudNest offers three plans: Free, Pro, and Team. Storage limits and feature availability differ by plan (see Pricing & Billing doc).

## Supported Platforms
- Desktop apps: Windows 10+, macOS 12+, Ubuntu 20.04+
- Mobile apps: iOS 15+, Android 10+
- Web app: any modern browser (Chrome, Firefox, Safari, Edge)
